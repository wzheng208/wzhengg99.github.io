Odin's Eye

Odin’s Eye provides live and retrospective monitoring of key metrics for 3 important pieces of a Kubernetes cluster — overall cluster metrics, individual pod and namespace metrics, and most importantly for this application, metrics for MongoDB databases deployed within a Kubernetes cluster.
Get Started

Get Started with Odin's Eye

Clone

Clone the Odin's Eye Repository

git clone https://github.com/oslabs-beta/OdinsEye.git

Port-Forward

Port-Forward your prometheus instance to port 9090.

kubectl port-forward -n 'prometheus' prometheus-prometheus-kube-prometheus-prometheus-0 9090

Npm

Npm install all the packages and npm start then go to http://localhost:3000/

npm install && npm start
